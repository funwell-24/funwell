The CSS Box Model

Every element in web design is a rectangular box. The CSS Box Model describes how these boxes are sized and spaced, consisting of: content, padding, borders, and margins.

Box Sizing Property

The box-sizing property controls how width and height are calculated:

content-box (default)

Width/height only apply to content. Padding and border are added.

border-box

Width/height include content, padding and border.
